# Pomodoro-Mac 番茄钟应用设计规格

## Context

帮助用户健康使用电脑的番茄钟应用。基于用户选择：菜单栏应用形态、丰富功能集（统计/目标/提醒/数据持久化）、Swift + AppKit 原生开发、温暖视觉风格（番茄红/橙色系）。

## 功能范围

### 核心计时
- **专注时长**：默认 25 分钟，可配置（15/20/25/30/45/60 分钟）
- **短休息时长**：默认 5 分钟，可配置（3/5/10 分钟）
- **长休息时长**：默认 15 分钟，每 4 个番茄周期后触发，可配置
- **番茄周期计数**：自动累计，完成长休息后重置

### 提醒系统
- **系统通知**：默认提醒方式，可设置是否带声音
- **弹窗提醒**：可配置为"长休息"时强制弹窗
- **休息结束提示音**：可选系统音效或静音

### 统计与目标
- **今日视图**：完成番茄数、累计专注时长、休息时长
- **目标设定**：每日番茄数目标（默认 8 个）
- **历史数据**：最近 7 天统计图表

### 数据持久化
- **存储位置**：UserDefaults（设置项）+ SQLite（统计数据）
- **数据保留**：最近 30 天记录

## 技术架构

### 平台形态
- **NSStatusItem**：菜单栏图标，常驻运行
- **NSPopover**：点击图标展开计时面板
- **LSUIElement = false**：保留 Dock 图标（可选设置为 true 完全隐藏）

- **SQLite.swift**：统计数据持久化

## 视觉规范

### 色彩
- **主色（番茄红）**：`#FF6347`
- **辅助色（暖橙）**：`#FFA07A`
- **背景色**：`#FFF5F5`（浅暖白）
- **文字主色**：`#2D2D2D`
- **文字次色**：`#6B6B6B`
- **成功色**：`#4CAF50`
- **休息色**：`#81C784`

### 字体
- **标题**：SF Pro Display，18pt，Semibold
- **计时器数字**：SF Pro Rounded，48pt，Bold
- **正文**：SF Pro Text，13pt，Regular
- **辅助文字**：SF Pro Text，11pt，Regular

### 圆角
- **面板圆角**：16pt
- **按钮圆角**：8pt
- **小元素圆角**：4pt

### 动画
- **计时器进度**：CABasicAnimation，圆环进度条
- **状态切换**：200ms ease-in-out 过渡
- **弹窗出现**：NSAnimationContext，默认系统动画

## 组件清单

### 1. 菜单栏图标
- **状态**：空闲（番茄轮廓）/ 专注中（实心番茄 + 进度环）/ 休息中（绿番茄）
- **尺寸**：18x18pt

### 2. 计时器弹窗（主面板）
- **尺寸**：320 x 400pt
- **顶部**：当前状态标签（专注中 / 休息中）
- **中部**：大号计时器数字 + 圆环进度条
- **底部**：开始/暂停按钮、重置按钮

### 3. 统计面板（Tab）
- **今日统计**：番茄数圆环图 + 时长
- **7日趋势**：柱状图

### 4. 设置面板（Tab）
- **时长配置**：专注/短休息/长休息时间选择器
- **提醒配置**：通知开关、弹窗开关、声音开关
- **目标配置**：每日番茄目标数

## 状态定义

| 状态 | 计时器颜色 | 图标样式 | 可用操作 |
|------|-----------|---------|---------|
| 空闲 | 灰色 | 番茄轮廓 | 开始 |
| 专注中 | 番茄红 | 实心番茄+进度环 | 暂停/重置 |
| 短休息 | 绿色 | 绿番茄 | 暂停/重置/跳过 |
| 长休息 | 深绿 | 绿番茄 | 重置/跳过 |
| 已暂停 | 当前状态色（半透明） | 暂停图标 | 继续/重置 |

## 系统集成

- **启动项**：可选开机自启动
- **菜单栏**：NSStatusItem 始终显示
- **通知**：UNUserNotificationCenter 授权
- **音效**：NSSound 系统音效
